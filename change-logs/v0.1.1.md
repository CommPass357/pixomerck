# Pixomerck v0.1.1

Web-only pivot.

- Removes the Android project and Gradle build from the repository.
- Keeps Pixomerck as a browser app served by the Windows FastAPI server.
- Adds browser login backed by a secure session cookie.
- Keeps `X-Pixomerck-Key` support for scripts and direct API clients.
- Publishes only the Windows server/web app zip and checksum.
