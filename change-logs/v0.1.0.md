# Pixomerck v0.1.0

First release of Pixomerck.

- Adds the Android app for prompt entry, CameraX capture, gallery selection, local person masking, LAN/tunnel server pairing, generation progress, and result sharing.
- Adds the Windows FastAPI generation server with invite-key auth, a one-job queue, ComfyUI integration, and PowerShell install/start/status/pair scripts.
- Publishes a debug-signed Android APK and Windows server zip with SHA256 checksums.
